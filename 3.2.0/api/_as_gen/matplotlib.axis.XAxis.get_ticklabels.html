matplotlib.axis.XAxis.get_ticklabels¶

XAxis.get_ticklabels(self, minor=False, which=None)¶

Get the tick labels as a list of Text instances.

Parameters:
minorbool

If True return the minor ticklabels, else return the major ticklabels

whichNone, ('minor', 'major', 'both')

Overrides minor.

Selects which ticklabels to return
Returns:
retlist

List of Text instances.
